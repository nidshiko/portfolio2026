Desbloqueados

Desbloqueados (Unblocked) — a board game designed to challenge gender stereotypes in early childhood education.

Desbloqueados was designed as a tool for ESI (Comprehensive Sexual Education) for preschool-age children. The brief was to create a game that actively challenges gender stereotypes — not by avoiding them, but by dismantling them through play.

The system is built from modular pieces combining geometric and irregular shapes. Most are double-sided, increasing the number of possible configurations. Some pieces include recognizable features — eyes, mouths, clothing — while others remain abstract. A third set is intentionally blank, paired with crayons, allowing children to create and define their own elements.

There are no fixed rules for assembly, only prompts. The goal is not to guide outcomes, but to remove constraints — making every combination equally valid.

The color palette is saturated and flat, designed to attract attention without suggesting meaning or hierarchy. The name Desbloqueados ("Unblocked") reflects this approach: opening possibilities rather than prescribing them.

The game was produced and tested in a real classroom setting. A preschool teacher introduced it to her students, and children engaged with it freely — extending the system by creating characters, narratives, and new rules of their own. That version — in use, reinterpreted, and slightly out of control — is the one that matters.
