Doce Hechizos

A visual system exploring how illustration can function consistently across different formats and scales.

Doce Hechizos (Twelve Spells) is a visual system exploring how illustration can function consistently across different formats and scales.

The brief was open, but the challenge was concrete: to build a complete system from scratch. The result is a Halloween-inspired universe applied across a 2026 calendar, stationery, and masks — all connected through a shared visual language.

The calendar became the central piece. I've designed calendars professionally for years, always within pre-existing systems. This project was the first time I built one from the ground up — from illustration to grid to typography — allowing for a level of control and expression rarely possible in commercial work.
